## Features
**Static site generation**
- Minified files
- SEO friendly
- Integrate external files

**Built-in markdown plugins**
- Table of contents
- Anchored Headings
- Code highlighting
- Callouts
- YouTube player embed
- Emoji

**Customization**
- Control theme color
- Dark theme :dark_sunglasses:
- Custom plugins support
